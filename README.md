# Игра "Попади в мишень"
Ваша цель - зарабатывать очки, стреляя по мишени

## Правила игры
- Выстрел происходит при клике мыши
- Разные цвета мишени дают разное количество очков. Желтый цвет прибавит Вам 5 очков, красный - 3, синий - 2, черный - 1
- Если Вы не попадаете по мишени, то отнимается одна жизнь
- Всего будет 3 жизни
- Через каждые 25 полученных очков переход на новый уровень
